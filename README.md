## About

By the end of this module, you'll have a solid understanding of intermediate Python concepts and be able to apply them effectively in your work.

## Content

| Lesson | Est. Delivery Time | Skills |
| ------ |:------------------:| ------ |
| [Setup](./setup/README.md)                                               |         10 min | Set up the development environment.                      |
| [Function Concepts](./function-concepts/README.md)                       |          5 min | Define the concept of a function in programming.         |
| [Functions and Modular Code](./functions-and-modular-code/README.md)     |         25 min | Using functions to make code reusable.                   |
| [Function Parameters](./function-parameters/README.md)                   |         20 min | Using parameters and understanding scope.                |
| [Modules and Packages](./modules-and-packages/README.md)                 |         20 min | Organizing code effectively and using modules.           |
| [Working With Files](./working-with-files/README.md)                     |         25 min | Utilizing modules and manipulating files to update data. |
| [Control Flow and Looping](./control-flow-and-looping/README.md)         |         20 min | Utilizing flow control and loops.                        |
| [Debugging and Error Handling](./debugging-and-error-handling/README.md) |         30 min | Learn how to handle errors and debug your code.          |
| **Total content**                                                         | 2 hours 35 min |                                                          |
